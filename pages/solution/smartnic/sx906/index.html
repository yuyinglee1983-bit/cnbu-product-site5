SmartNIC / DPU

SX906
Data Processing Unit

NVMe-oF Offload · IPsec/TLS Crypto · Isolated Management

Integrates network, security, and storage offloads directly into hardware, freeing host CPU cycles for business workloads. Supports NVMe-oF, IPsec/TLS, and OVS offload at line rate.

Dual 100/200G QSFP ARM Core Complex PCIe 5.0 x16 Crypto / IPsec Offload

Key Features

Full Hardware Network Offload

Offloads all packet processing—including OVS, SR-IOV, and RDMA—directly to the DPU silicon, eliminating host CPU involvement in the network data path.

IPsec / TLS Crypto Acceleration

On-chip AES-256-GCM and SHA-2 engines deliver line-rate encryption at 200 Gbps without consuming host CPU resources.

NVMe-oF Storage Offload

Native NVMe-oF target and initiator support with NVMe emulation allows the DPU to present remote storage as local devices with ultra-low latency.

Isolated Management Plane

The ARM processor subsystem runs an independent OS for out-of-band lifecycle management, telemetry, and zero-trust policy enforcement—fully isolated from the host workload.

PCIe 5.0 x16 Host Interface

Dual x8 bifurcation supports high-throughput host connectivity with minimal latency.

2× QSFP56 Network Ports

Up to 200 Gbps total bandwidth for high-density data center connectivity.

On-board ARM Processor

Independent ARM core complex runs isolated management OS for zero-trust control plane.
