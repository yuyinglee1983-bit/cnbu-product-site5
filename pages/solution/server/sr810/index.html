Key Features

Multi-GPU Acceleration Architecture

Supports up to four dual-slot PCIe 5.0 GPU accelerators for high-throughput deep learning training and AI inference, delivering maximum compute density in a 2U footprint.

Intel Xeon Scalable Dual-Socket

Dual Intel Xeon Scalable processors provide the CPU headroom required to feed multiple GPU accelerators, ensuring balanced compute performance across AI and HPC workloads.

High-Capacity DDR5 Memory

Up to 4 TB of DDR5 ECC registered memory delivers the bandwidth needed for large-model training datasets and in-memory analytics at scale.

CRPS Redundant Power Supply

3,200W Titanium-rated CRPS power supply with N+1 redundancy ensures continuous operation for mission-critical GPU workloads with no single point of power failure.
