SR610
1U DC-MHS Server with Dual Intel® Xeon® 6 Processors

SR610 is a 1U compact modular server built on the Intel® Xeon® 6 platform and OCP DC-MHS standard. It supports up to 12x NVMe drives and 2x PCIe Gen5 slots, making it ideal for virtualization, AI workload, and private cloud deployments. The CPU offers of P-core/E-core architecture, efficient airflow design, and centralized EDCC management, SR610 delivers outstanding compute density and flexibility for modern data centers.

Dual Intel® Xeon® 6 12× NVMe PCIe Gen5 DC-MHS

Key Features

Dual AMD EPYC 9004 Platform

Up to 192 cores / 384 threads with AVX-512 vector instruction support, designed for large-scale OLAP databases and HPC simulation workloads.

24-Bay All-NVMe Storage

24 NVMe U.3 bays with full PCIe 5.0 support, delivering sequential read speeds exceeding 200 GB/s+ for data-intensive workloads.

High-Capacity DDR5-4800 Memory

Up to 6 TB DDR5-4800 ECC RDIMM with 36% higher memory bandwidth than the previous generation, ideal for large-scale in-memory databases and computation.

Redfish Intelligent Management

IPMI 2.0 + OpenBMC + Redfish API with out-of-band management, centralized monitoring, and configurable alerting policies for enterprise-grade reliability.
